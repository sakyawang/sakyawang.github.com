Ubuntu18.4下搭建latex环境

发表于 2018-11-26 | 分类于 latex
Ubuntu18.4安装Tinytex，使用vscode+latex workshop搭建latex环境

环境

  • Ubuntu18.4
  • Vscode + latex workshop
  • TinyTeX

原本打算安装latex live但是安装需要5G存储空间,后来看到轻量级的TinyTeX，所以放弃使用Latex Live转而使用TinyTeX。

安装

  • VSCode 安装配置自行谷歌

  • Latex Workshop 直接在VSCode插件安装配置

  • TinyTeX安装

  • TinyTeX添加到path

  • 安装xelatex支持中文

使用

  1. 新建hello.tex
  2. 添加内容
# latex # vscode
